| 命令                | 作用                                 |
| ------------------- | ------------------------------------ |
| mv test.txt wbk.txt | 将test.txt重命名为wbk.txt            |
| mv /usr/udt/* .     | 将/usr/udt中的所有文件移动到当前目录 |
| sudo su             | 进入root                             |
| sudo passwd root    | root模式下修改root密码               |
| poweroff            | 关机                                 |
