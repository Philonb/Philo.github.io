| 命令                                       | 作用                           |
| ------------------------------------------ | ------------------------------ |
| xrandr -o normal                           | 恢复默认方向                   |
| xrandr -o left                             | 左旋转                         |
| xrandr -o q                                | 查看当前显示器的输出信息       |
| xrandr --output HDMI1 --rotate left/normal | 旋转某个视频接口的视频输出方向 |
| xrandr --help                              | 帮助                           |
| xrandr -s  2560x1600                       | 将分辨率修改为2560x1600        |
