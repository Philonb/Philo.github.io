| 命令                                      | 作用                           |
| ----------------------------------------- | ------------------------------ |
| xrandr -o normal                          | 恢复默认方向                   |
| xrandr -o left                            | 左旋转                         |
| xrand -o q                                | 查看当前显示器的输出信息       |
| xrand --output HDMI1 --rotate left/normal | 旋转某个视频接口的视频输出方向 |
| xrand --help                              | 帮助                           |
